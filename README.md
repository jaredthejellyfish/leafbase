## Usage

Set these variables in the .env file:

```.env
NEXT_PUBLIC_SUPABASE_URL=""
NEXT_PUBLIC_SUPABASE_ANON_KEY=""

OPENAI_API_KEY=""

KV_REST_API_READ_ONLY_TOKEN=""
KV_REST_API_TOKEN=""
KV_REST_API_URL=""
KV_URL=""

GROQ_API_KEY=""
```

Then, run the development server:

```bash
bun run dev
# or
yarn dev
# or
pnpm dev
```
